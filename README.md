## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Zaqueu Soares. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementaÃ§Ã£o de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon S3 (Simple Storage Service)
- Foco: Armazenamento escalável e seguro de dados
- Descrição de caso de uso: Migração de documentos regulatórios, laudos médicos, fichas técnicas de medicamentos e resultados de pesquisas clínicas para o S3, eliminando a necessidade de servidores físicos de armazenamento. Com o S3 Intelligent-Tiering, os arquivos menos acessados são automaticamente movidos para camadas de menor custo, reduzindo gastos com storage em até 70%.

Etapa 2: 
- Amazon EC2 com Auto Scaling
- Foco: Infraestrutura computacional elástica e sob demanda
- Descrição de caso de uso: Implementação de instâncias EC2 para hospedar sistemas de gestão farmacêutica (ERP), controle de estoque e rastreabilidade de medicamentos. Com Auto Scaling, a capacidade se ajusta automaticamente à demanda, pagando apenas pelos recursos utilizados. Substituição de servidores on-premise reduz custos com manutenção, energia elétrica e refrigeração em aproximadamente 60%.

Etapa 3: 
- Amazon RDS (Relational Database Service)
- Foco: Banco de dados relacional totalmente gerenciado
- Descrição de caso de uso: Migração dos bancos de dados de controle de lotes, registro de pacientes, histórico de vendas e inventário de medicamentos para o RDS. O serviço gerenciado elimina custos com administração de banco de dados, backups automatizados, patches de segurança e alta disponibilidade, reduzindo custos operacionais de TI em até 50% e garantindo conformidade com regulamentações como LGPD e ANVISA.



## Conclusão
A implementação de ferramentas na empresa *Abstergo Industries tem como esperado redução de custos operacionais de infraestrutura em até 65%, eliminação de gastos com manutenção de hardware, maior escalabilidade, segurança aprimorada e conformidade regulatória*, o que aumentará a eficiência e a produtividade da empresa. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
